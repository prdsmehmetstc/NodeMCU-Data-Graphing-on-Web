# NodeMCU-Data-Graphing-on-Web
## What is this?
A project to graph MPU6050 and NTC data via a web page running on the NodeMCU ESP8266.

## Used parts list
- NodeMCU DevKit ESP8266
- MPU6050 Gyro 
- 10k NTC
- 10k Resistor
